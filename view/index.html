GenCode API Documentation

POST /register

Registers a new user.

Request: 
{
  "username": "karan",
  "email": "karan@example.com",
  "password": "securepass"
}
Responses:

201 Created

{ "message": "User successfully registered." }

400 Bad Request

{ "message": "All fields (username, password, email) are required." }

409 Conflict

{ "message": "Username already exists." }

POST /login

Logs in the user and returns access token.

{
  "username": "karan",
  "password": "securepass"
}

202 Accepted

{ "accessToken": "..." }

400 Bad Request

{ "message": "Incomplete Fields" }

401 Unauthorized

{ "message": "Username does not exist" }

GET /refresh

Refreshes access token using cookie.

200 OK

{ "message": "Access token refreshed successfully.", "accessToken": "..." }

401 Unauthorized / 403 Forbidden

{ "message": "Unauthorized: No refresh token provided." }

GET /logout

Logs out the user.

200 OK

{ "message": "Logout successful." }

404 Not Found

{ "message": "User not found. Session cleared." }

POST /generate

Generates a competitive programming question using AI.

{
  "input": "string manipulation problem"
}

200 OK

{ "id": "generated_question_id" }

400 / 401

{ "message": "Missing prompt or unauthorized." }

GET /question

Returns all questions created by the user.

200 OK

{ "message": "Questions retrieved successfully.", "questions": [...] }

401 / 404

GET /question/:id

Returns a single question by ID.

200 OK

{ "message": "Question retrieved successfully.", "question": {...} }

404 Not Found

POST /run

Runs the user's code for a question.

{
  "language": "python",
  "code": "print('Hello')",
  "questionID": "..."
}

200 OK

{
  "message": "Code executed successfully.",
  "results": [
    {
      "input": "...",
      "expected": "...",
      "stdout": "...",
      "status": "Accepted",
      "passed": true
    }
  ]
}

401 / 404 / 500

POST /submit

Submits code for evaluation and updates question status.

{
  "language": "cpp",
  "code": "int main(){}",
  "questionID": "..."
}

200 OK

{
  "message": "Submission processed.",
  "results": [ ... ]
}

POST /submit/all

Returns all submissions for a question by the user.

{ "questionID": "..." }

200 OK

{ "message": "Submissions retrieved.", "submissions": [...] }

GET /stats/:username

Returns stats for a user (solved questions, language usage).

200 OK

{
  "message": "User stats retrieved successfully.",
  "stats": {
    "totalQuestions": 10,
    "solvedQuestions": 6,
    "languagesUsed": {
      "C": 1, "Cpp": 4, "Java": 0, "Python": 1
    }
  }
}

GET /

Default root route.

200 OK

{ "message": "Welcome to GenCode API." }
